# blog--SSM

* 使用 Maven 管理项目；
* 使用 Spring4+Springmvc+Mybatis3 架构；
* 采用 Mysql 数据库；
* 前端采用Bootstrap框架设计；
* 数据库连接池c3p0；
* 在线编辑器使用了Simditor富文本编辑器；
* 评论插件使用的多说。
* RESTful风格
***
#####tips:#####
1. 没有建立用户表，后台登录可以直接写在controller判断,也可以自行建表,相关页面需判断cookie；
2. 评论插件相关URL需修改，详情登录多说官网查看手册；
3. 本项目旨在学习使用SSM框架，包括相关配置、集成以及基本功能的实现。
***
